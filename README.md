## Установка  
1. Клонируйте репозиторий:  
   ```bash
   git clone https://github.com/kseniaxen/VN-data-analysis.git
   ```  
2. Установите зависимости:  
   ```bash
   pip install -r requirements.txt
   ```  
   (или `conda env create -f environment.yml`).  

## Использование  
Запустите Jupyter Notebook:  
```bash
jupyter notebook
```
